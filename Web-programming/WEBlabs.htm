Лабораторні роботи "Програмування та підтримка веб-застосувань"

Лабораторна робота №1

Створення простої гіпертекстової сторінки

Розробити просту гіпертекстову сторінку, яка буде містити матеріал лекції по даному предмету. Використати мінімальну кількість гіпертекстових тегів. Створити посилання на додаткові джерела інформаціх по даних питаннях, які можете знайти на сайті http://www.w3schools.com/. Можливі теми:

  1. Об'єктна модель документа HTML
  2. Форми. Технологія клієнт-сервер.
  3. Об'єкт screen.
  4. Об'єкт navigator.
  5. Об'єкт window.
  6. Об'єкт document.
  7. Об'єкт image.
  8. Основи SVG.
  9. Об'єкт image.
  10. Основи CGI.
  11. Обробка подій в HTML документі.
  12. Веб-сокети.
  13. Способи взаємодії з сервером.
  14. Створення мультиплікації засобами JavaScript.
  15. Бінарні типи даних в мові Javascript.
  16. Технологія AJAX.
  17. Об'єкт FileReader.
  18. Виконання скриптів за допомогою WebWorker.

Лабораторна робота №2

Розробка документа з фреймами (iframe)

Створити гіпертекстову сторінку з трьома фреймами. У верхній фрейм помістити назву даного курсу. В лівий фрейм помістити список лабораторних робіт. В правому фреймі повинно з'являтись завдання на відповідну роботу, що буде вибрана в лівому фреймі.

Лабораторна робота №3

Розробка власної таблиці стилів.

Для гіпертекстової сторінки сторінки, що була створена в першій лабораторній роботі добавити каскадну таблицю стилів. При здачі лабораторної роботи слід вміти змінювати стилі елементів у відповідності до вимог викладача

Лабораторна робота №4

Взаємодія скриптів з елементами гіпертексту.

Створити гіпертекстову сторінку, яка буде включати декілька обзаців та одне растрове зображення. Створити за допомогою скриптів зміну стилів абзаців при наведенні графічного курсора. При наведенні миші на графічне зображення воно повинно збільшуватись удвічі. Графічне зображення вставити в документ (BASE64)

Лабораторна робота №5

Написання CGI-скрипта. Обробка форм.

Створити гіпертекстову сторінку з формою, куди помістити всі елементи, які дозволяють ввід даних. За допомогою скрипта забезпечити контроль даних перед відправкою їх на сервер.

Для обробки даних, які переслані на сервер, створити CGI-програму. Розкодовані дані повернути клієнтові

Лабораторна робота №6

Написання PHP-скрипта. Робота з базою даних MySQL.

Написати PHP-скрипт для авторизації клієнта. Дані про клієнтів зберігати в таблиці MySQL або SQLITE.
