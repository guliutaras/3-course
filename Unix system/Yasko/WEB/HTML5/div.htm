HTML 5 теги, які створюють блок або секцію документа

Теги <div> </div>

Теги <div> </div> визначають секцію документа:

<html>
<body>
<div align="center">
<h4>HTML 5 теги</h4>
<p>У п'ятій версії HTML з'явились нові теги...</p>
</div>
</body>
</html>
