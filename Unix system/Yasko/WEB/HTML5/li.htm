HTML 5 теги, що визначають пункт списку

Теги <li> </li>

Теги <li> </li> визначають елемент (пункт) списку:

<html>
<body>
<ul>
<li>Sony</li>
<li>Toshiba</li>
<li>Asus</li>
</ul>
</body>
</html>

Результат:

  • Sony
  • Toshiba
  • Asus

Теги <li> </li> є елементами тегів <ol> </ol>, <ul> </ul>, <menu> </menu>, що визначають впорядковані і невпорядковані списки.
