HTML 5 теги, що визначають блок з цитатою

Теги <blockquote> </blockquote>

Теги <blockquote> </blockquote> визначають блок з цитатою:

<!DOCTYPE HTML>
<html>
<body>
<p>Оскар Уайльд, цитати:</p>
<blockquote>
«На древньогрецькому порталі античного світу було накреслено: пізнай самого себе. На порталі сучасного світу буде накреслено: будь самим собою»...
</blockquote>
<p>Зверніть увагу на характерні для <code>&lt;blockquote> &lt;/blockquote></code> відступи.</p>
</body>
</html>

Результат:

Оскар Уайльд, цитаты:

«На древньогрецькому порталі антічного світу було накреслено: пізнай самого себе. На порталі сучасного світу буде накреслено: будь самим собою»...

Зверніть увагу на характерні для <blockquote> </blockquote> відступи.
