HTML 5 тег, що впроваджує об'єкт у вигляді музики або відео

Тег <object />

Тег <object /> впроваджує в документ зображення, аудіо, відео, ActiveX, PDF та Flash файли:

<!DOCTYPE HTML>
<html>
<body>
<object data="golf.swf" width="240" height="320" />
</body>
</html>

Результат: HTML object

Атрибути та значення

  • Атрибут data="" – вказує url об'єкта.
  • width="" – ширина об'єкта.
  • height="" – висота об'єкта.

Деякі атрибути тега <object /> из HTML 4.01 не підтримуються в HTML 5.
