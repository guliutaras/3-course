HTML 5 тег, що визначає перенесення рядка на новий ряд

Тег <br />

Тег <br /> виконує перенесення рядка:

<!DOCTYPE HTML>
<html>
<body>
<p>Тег &lt;br /> <br /> розбиває рядок.</p>
</body>
</html>

Результат:

Тег <br />
розбиває рядок.

Ще приклад:

<p>Два тега, що йдуть один за одним &lt;br /> <br /><br /> збільшать відстань між рядками по вертикалі.</p>

Результат:

Два тега. що йдуть один за одним <br />

збільшать відстань між рядками по вертикалі.

Але це не привід використовувати їх в таких цілях! Для збільшення відстані користуйтесь CSS.
