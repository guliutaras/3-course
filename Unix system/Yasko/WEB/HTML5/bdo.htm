HTML 5 теги, що визначають напрямок тексту

Теги <bdo> </bdo>

Теги <bdo> </bdo> визначають напрямок тексту (зліва направо та справа наліво):

<!DOCTYPE HTML>
<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=windows-1251" />
</head>
<body>
<bdo dir="rtl"> Текст надрукований зліва направо, але відображений справа наліво. </bdo>
</body>
</html>

Результат:

Текст напечатан зліва направо, але відображений справа наліво.

Атрибути та значення

  • Атрибути dir="rtl", dir="ltl" – визначають напрямок тексту. В першому випадку напрямок буде справа наліво, а в другому – зліва направо (за замовчуванням).
  • title="" – вспливаюча підказка.
