HTML 5 тег, що визначає свойства колонок таблиці

Тег <col />

Тег <col /> визначає свойства колонок таблиці та їх вмісту:

<body>
<table border="1">
<col span="2" style="background-color:YellowGreen" />
<tr>
<th>Марка ноутбука</th>
<th>Виробник</th>
<th>Ціна</th>
</tr>
<tr>
<td>Sony</td>
<td>Японія</td>
<td>1400 у.е.</td>
</tr>
<tr>
<td>Toshiba</td>
<td>Японія</td>
<td>1200 у.е.</td>
</tr>
</table>
</body>

Результат:

Марка ноутбука Виробник Ціна
Sony Японія 1400 у.е.
Toshiba Японія 1200 у.е.

Атрибуты та значення

  • span="2" – форматування поширюється на дві колонки.
  • style="" – лінійне CSS включення.
