HTML 5 теги, які дозволять створити картинку всередині сторінки

Теги <svg> </svg>

Тег <svg> </svg> (англ. svg — «Scalable Vector Graphics») — призначений для створення векторного зображення за допомогою спеціальних тегів в форматі XML, які можна використовувати тільки всередині даного елементу.

<html> <head> <meta charset="UTF-8"> <title>Приклад використання</title> </head> <body> <svg xmlns="http://www.w3.org/2000/svg" version="1.1"> <circle cx="100" cy="50" r="40" stroke="black" stroke-width="2" fill="red"/> </svg> </body> </html>
