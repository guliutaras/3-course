HTML 5 теги, створюючі великі осередки всередині таблиці

Теги <th> </th>

Теги <th> </th> визначають комірку із заголовком:

<html>
<body>
<table border="1">
<tr>
<th>Перша комірка першого ряду</th><th>Друга комірка першого ряду</th>
</tr>
<tr>
<td>Перша комірка другого ряду</td><td>Друга комірка другого ряду</td>
</tr>
</table>
</body>
</html>

Результат:

Перша комірка першого рядуДруга комірка першого ряду
Перша комірка другого рядуДруга комірка другого ряду
