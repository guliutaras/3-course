HTML 5 теги, що визначають заголовки

Теги <h1> </h1> - <h6> </h6>

Теги <h1> </h1> - <h6> </h6> визначають заголовки всередині web-сторінки:

<html>
<body>
<h5>Заголовок п'ятого рівня</h5>
<h4>Заголовок четвертого рівня</h4>
<p>Параграф</p>
</body>
</html>
