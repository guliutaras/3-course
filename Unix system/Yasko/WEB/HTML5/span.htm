HTML 5 теги, визначають лінійний відсік

Теги <span> </span>

Теги <span> </span> определяют участок текста. Используются при форматировании:

<html>
<body>
<p>ділянка тексту <span style="color:#ff0000">червоного</span> кольору.</p>
</body>
</html>

Результат:

ділянка текстучервоного кольору.

Самі по собі теги<span> </span> ніяк не впливають на спосіб відображення контенту. Зазвичай вони вбудовуються в блокові елементи, такі як<div> </div> або <p> </p>, але не навпаки.Теги <span> </span> є контейнером для певної ділянки тексту. Використовуються при форматуванні тексту за допомогоюCSS.
