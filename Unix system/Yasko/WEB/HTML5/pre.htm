HTML 5 теги, визначають попереднє форматування

Теги <pre> </pre>

Теги <pre> </pre> визначають, попередньо відформатований у вихідному коді текст:

<html>
<body>
<pre> Теги відобразять текст з точною кількістю, проставлених
    в вихідному коді,     прогалин і переносів рядків.
До того ж, текст буде моноширінним.
Моноширинний &ndash; всі букви однакової ширини (машинописний шрифт).
</pre>
</body>
</html>

Результат:

Теги відобразять текст з точною кількістю, проставлених
     в вихідному коді,        прогалин і переносів рядків.
