HTML 5 тег, що вмикає інтерактивні об'єкти

Тег <embed />

Тег <embed /> впроваджує інтерактивний об'єкт у документ:

<html>
<body>
<embed src="mult.swf" />
</body>
</html>

Обов'язковий атрибут src="" вказує url об'єкта, яким може бути, наприклад, флеш ролік.

Тег <embed /> вперше представлено у специфікації HTML 5.
