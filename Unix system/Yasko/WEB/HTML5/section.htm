HTML 5 теги, що оформлюють секцію чи блок

Теги <section> </section>

Теги <section> </section> аналогічно <div> </div> визначають секцію (блок):

<html>
<body>
<section align="center">
<h4>HTML 5 теги</h4>
<p>В п'ятій версії HTML з'явились <i>нові теги</i>...</p>
<p> ............. </p>
</section> </body>
</html>

Результат:

HTML 5 теги

В п'ятій версії HTML з'явились нові теги...

.............

Теги <section> </section> вперше представлені в специфікаціїHTML 5.
