HTML 5 тег, що визначає поле вводу чи кнопку

Тег <input />

Тег <input /> створює поле для вводу даних чи кнопку:

<html>
<body>
<form action="">
<input type="radio" name="Sex" value="Чоловіча" checked="checked" /> Чоловік<br />
<input type="radio" name="Sex" value="Жіноча" /> Жінка<br />
<input type="text" size="20" value="Росія" /> Країна<br />
<input type="text" size="20" /> Місто<br /><br />
<input type="image" src="../images/input_img.png" /><br />
</form>
</body>
</html>

Атрибути і значення

  • Атрибут type="" зі значеннями: button, checkbox, color, date, datetime, datetime-local, email, file, hidden, image, month, number, password, radio, range, reset, search, submit, tel, text, time, url, week – вказує тип тегу <input /> (text – за замовчанням).
  • Атрибут src="" – вказує url зображення-кнопки, затосовується разом з type="image".
  • Прапорець checked="checked" – встановлює для прикладу одне з запропонованих значень, застосовується разом з type="checkbox" или type="radio".
  • Атрибут name="" – присвоює ім'я елементу. Застосовується для ідентифікації отриманої інформації при обробці даних.
  • Атрибут value="" використовується для різноманітних цілей: надписує кнопки (якщо застосовується разом з type="submit" або з type="reset"), зберігає деякі значення, що відсилаються на обробку (якщо застосовується разом з type="checkbox" або з type="radio"), встановлює значення по замовчуванню для текстових полів вводу даних (якщо застосовується разом з type="text" або з type="password").
  • Атрибут size="20" – встановлює довжину текстового поля в символах.
  • Прапорець required="required" – визначає обов'язкове для заповнення поле.
  • Атрибут autocomplete="" зі значеннями on, off – каже браузеру зберігати чи не зберігати введені дані в поля форми. Іншими словами, дані автоматично відобразяться в текстовому полі форми, при повторній спробі вводу, якщо встановлено значення on.
  • Прапорець multiple="multiple" – дозволяє вибрати більше одного значення зі списку.
  • Атрибут min="6" – встановлює мінімально допустиму кількість символів, що вводятьсяв.
  • Атрибут max="20" – встановлює максимально допустиму кількість символів, що вводяться.
  • Атрибут maxlength="30" – встановлює максимально допустиму кількість символів, що вводяться в текстове поле.
  • Атрибут height="" – визначає висоту поля в пікселях або відсотках.
  • Атрибут width="" – визначає ширину поля в пікселях або в відсотках.
  • Атрибут form="ім'я форми" – вказує ім'я форми, якій належить елемент <input />.
  • Атрибут formmethod="" зв значеннями get, post, put, delete – визначає метод відсилання даних на обробку по визначеному url. Має перевагу перед атрибутом method="" тега <form>.
  • Атрибут formaction="url" – визначає url, за яким буде відіслано дані. Має перевагу над атрибутом action="" тега <form>.
  • Атрибут list="" – дивіться теги <datalist> </datalist>.

Додатково дивіться HTML форми.
