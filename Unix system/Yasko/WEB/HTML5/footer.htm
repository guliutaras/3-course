HTML 5 теги, що визначають нижню секцію документа

Теги <footer> </footer>

Теги <footer> </footer> визначають нижню частину документа або секції:

<!DOCTYPE HTML>
<html>
<body>
<footer>Дата публікації: Лютий 2010</footer>
</body>
</html>

Теги <footer> </footer> можуть містити дату, им'я автора, контактну інформацію.

Теги <footer> </footer> вперше представлені у специфікації HTML 5.
