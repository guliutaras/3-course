HTML 5 теги, що формують навігаційне меню

Теги <nav> </nav>

Теги <nav> </nav> що визначають навігаційне меню:

<html>
<body>
<nav>
<a href="#">Попередня сторінка</a> <a href="#">Наступна сторінка</a>
</nav>
</body>
</html>

Результат:

Попередня сторінка Наступна сторінка
