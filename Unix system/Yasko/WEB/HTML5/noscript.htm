HTML 5 теги, що відображають попереджуюче повідомлення

Теги <noscript> </noscript>

Теги <noscript> </noscript> містять повідомлення, яке буде показано в браузері клієнта якщо у нього вимкнена підтримка скриптів:

<html>
<body>
<script type="text/javascript">
..........
</script>
<noscript>Ваш браузер не підтримує JavaScript!</noscript>
</body>
</html>
