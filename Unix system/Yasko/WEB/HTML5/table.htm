HTML 5 теги, визначають таблиці

Теги <table> </table>

Теги <table> </table> визначають таблицю:

<html>
<body>
<table border="2">
<tr>
<td>Перша комірка першого ряду</td><td>Друга комірка першого ряду</td>
</tr>
<tr>
<td>Перша комірка другого ряду</td><td>Друга комірка другого ряду</td>
</tr>
</table>
</body>
</html>

Результат:

Перша комірка першого рядуДруга комірка першого ряду
Перша комірка другого рядуДруга комірка другого ряду

Атрибут border="" – встановлює ширину рамки таблиці.
