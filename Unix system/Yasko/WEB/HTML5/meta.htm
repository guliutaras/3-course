HTML 5 тег, що містить службову інформацію про документ

Тег <meta />

Тег <meta /> містить стислу інформацію про документ, інструкції по інтерпретіції коду:

<html>
<head>
<meta name="keywords" content="HTML довідник мета-теги" />
<meta name="author" content="Вася Мітін" />
<meta http-equiv="Content-Type" content="text/html; charset=windows-1251" />
<meta http-equiv="pragma" content="no-cache" />
<meta name="revisit" content="12" />
<meta name="robots" content="index, follow" />
</head>
<body>
</body>
</html>
