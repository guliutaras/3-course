HTML 5 теги, що включають скрипти

Теги <script> </script>

Теги <script> </script> визначають скрипт:

<html>
<body>
<script type="text/javascript">
..........
</script>
</body>
</html>

Теги <script> </script> є контейнером для сценарія, що виконується на стороне клієнту, тобто в браузері користувача (Javascript, Vbscript), без використання серверу.

Атрибути і значения

  • Атрибут type="" – вказує тип скрипту. Типи: text/ecmascript, text/javascript, application/ecmascript, application/javascript, text/vbscript.
  • Прапорецьasync="async" – визначає чи буде скрипт виконаним одночасно з завантаженням сторінки чи ні.
  • Прапорецьdefer="defer" – каже браузеру, що якщо скрипт і буде виконаний, то тільки після загрузки web-сторінки.
  • Атрибут src="http://..." – визначає url зовнішнього файлу зі сценарієм.
    • Якщо позначений прапорець async="async", скрипт буде виконаний одночасно з загрузкою web-сторінки.
    • Якщо позначений прапорець defer="defer", скрипт буде виконаним після завантаження сторінки.
    • Якщо прапорці не позначені, скрипт буде виконаним або активованим перед тим, як відбудеться повне завантаження web-сторінки.
