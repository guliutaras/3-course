HTML 5 теги, що визначають початкову частину сторінки

Теги <header> </header>

Теги <header> </header> визначають початок документа, його заголовок:

<html>
<head>
</head>
<body>
<header>
<h2>Сучасні уроки по HTML</h2>
<p>HTML 5 – нова версія Гіпертекстової мови розмітки документів...</p>
</header>
...........
</body>
</html>

Теги <header> </header> вперше представлені у специфікації HTML 5.
