HTML 5 теги, які групують заголовки

Теги <hgroup> </hgroup>

Теги <hgroup> </hgroup> визначають групу заголовків:

<html>
<body>
<hgroup>
<h1>HTML довідник</h1>
<h4>Нові теги</h4>
</hgroup>
<p>У специфікації HTML 5 представлені нові теги...</p>
</body>
</html>

Теги <hgroup> </hgroup> вперше представлені у специфікації HTML 5.
