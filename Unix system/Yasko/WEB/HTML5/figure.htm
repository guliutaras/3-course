HTML 5 теги, що групують елементи сторінки

Теги <figure> </figure>

Теги <figure> </figure> групують елементи web-сторінки за змістом:

<!DOCTYPE HTML>
<html>
<body>
<figure>
<h4>W3C</h4>
<p>W3C &ndash; oрганізація, що розробляє та впроваджує технологічні стандарти для Всесвітнього павутиння...</p>
</figure>
</body>
</html>

Результат:

Теги <figure> </figure> вперше представлені у специфікації HTML 5.
