HTML 5 теги, що маркерують важливу частину тексту

Теги <mark> </mark>

Теги <mark> </mark> визначають важливу по смислу частину тексту:

<html>
<body>
<p>В <b>HTML 5</b> введені <mark>нові теги</mark>.</p>
</body>
</html>

Результат:

В HTML 5 введені нові теги.

Теги <mark> </mark> ніяк не форматують текст, більш за все, таке маркування використовується для пошукової оптимізації.

Теги <mark> </mark> вперше представлені в специфікаціїHTML 5.
