HTML 5 теги, що визначають форми

Теги <form> </form>

Теги <form> </form> визначають HTML форми:

<html>
<body>
<form action="URL" method="GET|POST">
<input type="radio" name="Sex" value="Чоловічий" checked="checked" /> Чоловік<br />
<input type="radio" name="Sex" value="Жіночий" /> Жінка<br />
<input type="text" size="20" /> Вік<br />
<input type="text" size="20" /> Країна<br />
<input type="submit" value="Надіслати" /><br />
</form> </body>
</html>
