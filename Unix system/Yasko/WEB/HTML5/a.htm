HTML 5 теги, визначають гіперпосилання

Теги <a> </a>

Теги <a> </a> визначають гіперпосилання:

<!DOCTYPE HTML>
<html>
<body>
<p><a href="#top">Гиперпосилання</a></p>
</body>
</html>

Результат:

Гіперпосилання

або закладку (якір - anchor) всередині документа, на яку посилається гіперпосилання: <a name="top">...</a>

За замовчуванням колір гіперпосилання синій, відвіданою –пурпурний, активної – червоний.

Дивіться урокHTML гиперпосилання.
