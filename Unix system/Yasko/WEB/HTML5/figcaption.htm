HTML 5 теги, задающие заголовок элемента <figure> </figure>

Теги <figcaption> </figcaption>

Теги <figcaption> </figcaption> группируют элементы web-страницы по смыслу:

<!DOCTYPE HTML>
<html>
<body>
<figure>
<figcaption> Заголовок </figcaption> <h4>W3C</h4>
<p>W3C &ndash; oрганизация, разрабатывающая и внедряющая технологические стандарты для Всемирной паутины...</p>
</figure>
</body>
</html>

Результат:

Теги <figure> </figure> впервые представлены в спецификации HTML 5.
