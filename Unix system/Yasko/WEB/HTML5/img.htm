HTML 5 тег, що визначає зображення

Тег <img />

Тег <img /> визначає зображення:

<html>
<body>
<img src="../images/porshe.jpg" hspace="20" alt="" />
</body>
</html>

Результат:

Атрибути й значення

  • src="../images/porshe.jpg" – вказує url зображення.
  • hspace="20" – встановлює віжступ від зображення по горизонталі в 20 пікселів.
