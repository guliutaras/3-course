HTML 5 теги, що визначають команди для кнопок або списків

Теги <command> </command>

Теги <command> </command> відображають команди для випадаючих списків, кнопок:

<!DOCTYPE HTML>
<html>
<body>
<menu>
<command type="command">Виберіть...</command>
</menu>
</body>
</html>

Теги <command> </command> повинні бути розміщені всередині <menu> </menu>.

Результат:

Виберіть...
