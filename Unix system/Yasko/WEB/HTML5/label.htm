HTML 5 теги, що визначають маркер, наприклад, для форми

Теги <label> </label>

Теги <label> </label> створюють маркер (лэйбу) для елементів форми:

<html>
<body>
<form action="">
<input type="radio" name="answer" value="yes" />
<label>Да</label>
<br />
<input type="radio" name="answer" value="no" />
<label>Ні</label>
</form>
</body>
</html>
