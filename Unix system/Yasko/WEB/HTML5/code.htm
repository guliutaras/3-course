HTML 5 теги, що визначають моноширинний текст

Теги <code>, <kbd>, <samp>

Теги <code>, <kbd>, <samp> перетворюють шрифт в моноширинний:

<body>
<p>Теги, які перетворюють текст в <code>моноширинний</code>.</p>
<p>Теги, які перетворюють текст в <kbd>моноширинний</kbd>.</p>
<p>Теги, що перетворюють текст в <samp>моноширинний</samp>.</p>
</body>

Моноширинний – всі букви одинакової ширини (машинописний).
