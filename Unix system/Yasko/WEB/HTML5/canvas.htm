HTML 5 теги, які дозволять створити картинку всередині сторінки

Теги <canvas> </canvas>

Тег <canvas> </canvas> (англ. canvas — «холст») — призначений для створення растрового зображення за допомогою JavaScript.

<html> <head> <title>Приклад використовування</title> </head> <body> <canvas id='example'> Цей текст відобразиться, якщо браузер не підтримує html5 елемент canvas. </canvas> <script type="application/javascript"> var canvas = document.getElementById("example"); var context = canvas.getContext("2d"); function drawCircle(x, y, r) { context.arc(x, y, r, 0, 2 * Math.PI, false); } context.beginPath(); drawCircle(150, 75, 50); context.lineWidth = 15; context.strokeStyle = "#0f0"; context.stroke(); </script> </body> </html>
