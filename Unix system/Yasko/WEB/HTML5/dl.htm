HTML 5 теги, що визначають список визначень

Теги <dl> </dl>

Теги <dl> </dl> визначають список визначень:

<dl>
<dt>Австралія</dt>
<dd>Шостий за розміром материк</dd>
<dt>Мадагаскар</dt>
<dd>Острів у західній частині Індийського океана</dd>
</dl>
