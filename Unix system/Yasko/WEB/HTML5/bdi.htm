HTML 5 теги, які задають напрям тексту

Теги <bdi> </bdi>

Теги <bdi> </bdi> визначають фрагмент тексту, який буде використовуватись для двух напрямів:

<ul> <li>User <bdi>jcranmer</bdi>: 12 posts. <li>User <bdi>hober</bdi>: 5 posts. <li>User <bdi>إيان</bdi>: 3 posts. </ul>

Результат:

  • User jcranmer: 12 posts.
  • User hober: 5 posts.
  • User إيان: 3 posts.
