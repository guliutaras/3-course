HTML 5 теги, перетворюючі списки в меню

Теги <menu> </menu>

Теги <menu> </menu> визначають список-меню:

<html>
<body>
<menu>
<li><input type="checkbox" /> Так</li>
<li><input type="checkbox" /> Ні</li>
<li><input type="checkbox" /> Не знаю</li>
</menu>
</body>
</html>

Результат:

  • Так
  • Ні
  • Не знаю

В HTML 4.01 теги <menu> </menu> були заборонені.
