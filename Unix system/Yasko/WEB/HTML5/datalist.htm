HTML 5 теги, що визначають деякі допустимі значення

Теги <datalist> </datalist>

Теги <datalist> </datalist> визначають допустимі значення при введенні даних:

<!DOCTYPE HTML>
<html>
<body>
<input list="computers" />
<datalist id="computers">
<option value="sony" />
<option value="toshiba" />
<option value="asus" />
<option value="acer" />
</datalist>
</body>
</html>

Результат:

Теги <datalist> </datalist>, З обов'язковими в даному випадку <option />, не видимі в браузері, вони лише визначають допустимі значення при введенні даних за допоиогою елемента <input />. Атрибут list="" тега <input /> вказує на функціональний зв'язок між групою тегів.

Теги <datalist> </datalist> вперше представлені в специфікації HTML 5.
