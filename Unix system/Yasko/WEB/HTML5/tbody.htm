HTML 5 теги, відокремлюючі тіло таблиці

Теги <thead>, <tfoot>, <tbody>

Теги <thead>, <tfoot>, <tbody> визначають секції таблиці:

<html>
<body>
<table border="1">
<thead>
<tr>
<th></th><th></th>
</tr>
</thead>
<tfoot>
<tr>
<td></td><td></td>
</tr>
</tfoot>
<tbody>
<tr>
<td></td><td></td>
</tr>
</tbody>
</table>
</body>
</html>

Теги <thead>, <tfoot>, <tbody> не використовуються окремо один від одного. Порядок розташування повинен дотримуватися– <tbody> розташовується внизу таблиці.

Дані теги не є обов'язковими і використовуються для логічної зв'язки вмісту в межах однієї таблиці.
