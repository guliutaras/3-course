HTML 5 теги, які показують результат обчислення

Теги <output> </output>

Теги <output> </output> показують результат обчислення:

<form onsubmit="return false" oninput="o.value = Number(a.value) + Number(b.value)"> <input name=a type=number value=1 step=any> + <input name=b type=number value=2 step=any> = <output name=o></output> </form>

Результат:

+ =
