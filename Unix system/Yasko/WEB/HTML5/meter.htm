HTML 5 тег, який показує вимірювання

Тег <meta />

Тег <meta /> показцє величину у відомому діапазоні або відсоток:

<html> <body> <pre> Storage space usage: <meter value=6 max=8>6 blocks used (out of 8 total)</meter> Voter turnout: <meter value=0.75><img alt="75%" src="graph75.png"></meter> Tickets sold: <meter min="0" max="100" value="75"></meter> </pre> </body>

Результат:

Storage space usage: 6 blocks used (out of 8 total)
Voter turnout:
