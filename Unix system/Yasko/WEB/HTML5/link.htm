HTML 5 тег, що визначає посилання на зовнішній файл

Тег <link />

Тег <link /> визначає посилання на зовнішнє джерело (файл):

<html>
<head>
<link rel="stylesheet" href="../style.css" type="text/css" />
</head>
<body>
</body>
</html>

Розміщується в секції <head> </head>.

Атрибути і значення

  • rel="stylesheet" – вказує тип зв'зку між документом і віддаленим джерелом.
  • href="../style.css" – вказує url джерела
  • type="text/css" – вказує тип зовнішнього джерела (файла).

Зазвичай тег <link /> містить посилання на віддалений .css файл (дивіться Впровадження CSS в HTML документ).
