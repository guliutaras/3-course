HTML 5 теги, що визначають службову секцію

Теги <head> </head>

Теги <head> </head> містять службову інформацію про документ:

<!DOCTYPE HTML>
<html>
<head>
<title>Основний заголовок HTML документа</title>
<meta http-equiv="Content-Type" content="text/html; charset=windows-1251" />
</head>
<body>
Видима частина документа
</body>
</html>
