HTML 5 теги, які задабть прогрес-бар

Теги <progress> </progress>

Теги <progress> </progress> задають прогрес-бар.

<html> <body> <section> <h2>Task Progress</h2> <p>Progress: <progress id="p" max=100><span>0</span>%</progress></p> <script> var progressBar = document.getElementById('p'); function updateProgress(newValue) { progressBar.value = newValue; progressBar.getElementsByTagName('span')[0].textContent = newValue; } </script> </section> </body> </html>
