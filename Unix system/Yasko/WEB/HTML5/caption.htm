HTML 5 теги, що визначають напис над таблицею

Теги <caption> </caption>

Теги <caption> </caption> визначають напис над таблицею:

<table border="1">
<caption>Прайс-лист ноутбуків</caption>
<tr>
<th>Марка</th>
<th>Країна виробник</th>
<th>Ціна</th>
</tr>
<tr>
<td>Sony</td>
<td>Японія</td>
<td>1400 у.е.</td>
</tr>
<tr>
<td>Toshiba</td>
<td>Японія</td>
<td>1200 у.е.</td>
</tr>
<tr>
<td>Acer</td>
<td>Китай</td>
<td>700 у.е.</td>
</tr>
</table>

Результат:

Прайс-лист ноутбуків
Марка Виробник Ціна
Sony Японія 1400 у.е.
Toshiba Японія 1200 у.е.
Acer Китай 700 у.е.
