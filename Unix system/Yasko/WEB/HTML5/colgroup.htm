HTML 5 теги, що узагальнюють колонки таблиці для форматування

Теги <colgroup> </colgroup>

Теги <colgroup> </colgroup> аналогічно непарному <col /> визначають свойства колонок таблиці та їх вмісту:

<body>
<table border="1">
<colgroup span="2" style="background-color:LightPink"> </colgroup>
<tr>
<th>Марка ноутбука</th>
<th>Виробник</th>
<th>Ціна</th>
</tr>
<tr>
<td>Sony</td>
<td>Японія</td>
<td>1400 у.е.</td>
</tr>
<tr>
<td>Toshiba</td>
<td>Японія</td>
<td>1200 у.е.</td>
</tr>
</table>
</body>

Результат:

Марка ноутбука Виробник Ціна
Sony Японія 1400 у.е.
Toshiba Японія 1200 у.е.
