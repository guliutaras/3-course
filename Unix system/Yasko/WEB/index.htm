Практикум по программированию на JavaScript

Темы занятий

  1. Основные понятия. Введение в JavaScript. Размещение сценария на Web-странице.
  2. Константы, переменные, типы данных. Запись констант. Работа с переменными. Примеры объектов.
  3. Операции. Типы операций.
  4. Оператор if и ветвление. Логическое разветвление программ. Условные операторы.
  5. Введение в циклы. Повторяющиеся действия. Циклы с предусловием.
  6. Функции в JavaScript. Их создание и использование.
  7. Массивы. Введение в массивы. Их создание и обработка.
  8. Объекты. Встроенные объекты. Дата и время. Математические вычисления.
  9. Регулярные выражения.
  10. Концепция свойств. Объектная модель документа HTML. DOM и dhtml. Основные объекты и их свойства.
  11. Формы. Технология клиент-сервер. Создание функций для обработки данных пользователя в формах. Предварительная обработка данных форм. Динамическое изменение содержимого форм.
  12. События в JavaScript. Обработка событий. (onload, onunload, onclick, onmouseover, onmouseout, onkeydown, onkeypress, onkeyup)
  13. Способы взаимодействия с сервером. Формы, AJAX и JSON.
  14. Ошибки в сценариях. Виды ошибок и методы их поиска. try/catch/finally.
  15. Динамическое изменение изображений с помощью функций. Динамическая загрузка и обновление изображений. Основы мультипликации в JavaScript.
  16. Рекомендации по Web-программироваю.
