ПП 1.4.15 Програмування та підтримка веб-застосувань

Факультет прикладної математики

Галузь знань: Інформаційні технології

Спеціальність 6.040302 Комп'ютерні науки та інформаційні технології

Освітньо-кваліфікаційний рівень: бакалавр


Мета вивчення дисципліни є набуття студентами знань щодо ознайомлення з історією побудови інформаційних систем їх специфікою, вивчення правил роботи глобальних інформаційних системах та набуття студентами практичних навичок щодо роботи в глобальній мережі Internet; використання глобального інформаційного простору задля пошуку потрібної інформації, розміщення своєї інформації в глобальній мережі та розв’язання бізнес завдань за допомогою глобальної мережі.

Завдання

  1. Вивчити основи мови HTML, CSS, Javascript, PHP.
  2. Засвоїти основні поняття взаємодії скриптів з об’єктами HTML.
  3. Знати як встановити WEB-сервер та розробляти програмне забезпечення для нього.

Змістовий модуль 1. Програмування на стороні клієнта.

  1. Вступ. Основи мови HTML5. Структура гіпертекстового документа.
  2. Основні теги HTML5. Форматування тексту. Заголовки, розділи, абзаци, списки.
  3. Каскадні таблиці стилів, рекомендації по їх використанню.
  4. Основи мови Javascript.
  5. Мова Javascript. Елементи гіпертекту як об’єкти Javascript.
  6. Об’єкти та формат JSON.
  7. Об’єктна модель DHTML та DOM. Рекомендації по їх використанню.
  8. Основи XML.
  9. Новітнє Javascript API.

Змістовий модуль 2. Програмування на стороні сервера.

  1. Конфігурування веб-сервера.
  2. Програмування форм. Види елементів вводу. Технологія клієнт-сервер.
  3. CGI-програми. Perl-скрипти.
  4. Способи організації взаємодії клієнт-сервер. AJAX.
  5. PHP-скрипти. Програмування на PHP. Робота з базою даних MySQL.
  6. Використання веб-сокетів. Node.js.

Теми лабораторних занять

  1. Створення гіпертекстового документа
  2. Використання каскадних таблиць стилів
  3. Розробка сценаріїв javascript
  4. Обробка форм. Створення CGI-програми на мові C/C++
  5. Створення CGI-програми на мові Perl
  6. Обробка даних із форм на мові PHP
  7. Робота з базою даних MySQL
  8. Формати XML та JSON
  9. Створення векторної графіки

Програмне забезпечення

  • ОС:

    Windows, Linux, Apple Mac

  • web browser:

    Mozilla Firefox, Microsoft Internet Explorer (IE11), Edge, Apple Safari, Google Chrome, Opera

  • web server:

    XAMPP ver. 1.8.3+ (Apache,Perl,PHP,MySQL,...), IIS, nginx,...

  • node.js (http://www.nodejs.org)
