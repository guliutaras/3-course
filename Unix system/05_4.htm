Лабораторна робота № 4

Завдання

Написати на мові Perl або bash скрипт, який прочитає зміст усіх файлів, що передані йому в командній лінії та об'єднає їх в один файл. Якщо серед переданих файлів будуть файли в форматі HTML, то в них необхідно знищити всі теги.

Текст розв'язку

#!/bin/bash out=$1 for i in $@ do if [ "${i##*.}" == "html" ]; then sed -r 's/<[^>]*>//g' $i >> $out else cat "$i" >> "$out" fi done

Пояснення

Програма має досить просту структуру. Спочатку перший параметр позначається як вихідний файл у який буде поєднано інші файли. Потім виконується цикл який перебирає усі значення параметрів у разі знаходження файлу з розширенням html виконуэться об'єднання лише тієї частини файлу яка не містить тегів у іншому випадку файл записується повністью. Перевірка і виокремлення виконується за допомогою регулярних виразів
